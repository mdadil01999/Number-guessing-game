Number Guessing Game
A simple and interactive browser-based number guessing game built using HTML, CSS, and JavaScript.

The player has 9 attempts to guess the correct number between 1 and 100.
The game provides hints, sound feedback, closest higher/lower guesses, and warning effects for the final attempts.

Features
Random number generation
Maximum 9 attempts
Attempt counter
Attempts left tracker
Closest lower guess display
Closest higher guess display
Warning effect during final attempts
Sound feedback using JavaScript Audio API
Restart game option
Responsive dark UI design
Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)

🎮 How to Play
Enter a number between 1 and 100
Click the "Submit Guess" button
The game will tell you if your guess is too high or too low
You only get 9 attempts
Win by guessing the correct number before attempts run out
